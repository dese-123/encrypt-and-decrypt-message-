🔐 How to Use the Crypt Message (Web-Based)
✅ Step 1: Create the file

Open Notepad (or any text editor)

Copy the crypt message code

Save it as:

crypt.html
✅ Step 2: Open in browser

Double-click crypt.html

It will open in Chrome / Edge / Firefox

You will see a button:

Create Crypt

✅ Step 3: Encrypt (Create Crypt Message)

Click Create Crypt

A popup appears:

Enter message to encrypt:

Example:

HELLO

Another popup appears:

Enter secret key (number):

Example:

3
✅ Step 4: View Crypt Message

You will see:

Crypt Message:
KHOOR

🎉 Your message is now encrypted (crypt message)
